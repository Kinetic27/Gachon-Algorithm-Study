## 1. 백준이 뭐임?
문제양이 상당히 많은 무료 코딩 문제 은행 사이트야, 난이도별로 많으니 수준에 맞춰서 공부할 수 있어  
코딩 고수님들이 주로 상주하고, 또 질문을 올릴 경우 이분들이 답변을 잘해주셔  

테스트 데이터가 많아서 정확한 코드 채점을 받아볼 수 있어  
그리고 그룹을 만들어 우리끼리 가상의 대회를 열어볼 수도 있어!!

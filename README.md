문제 잘 읽고 제출탭 가셔서 언어 선택하시고 정답코드 제출하세요.
입력이 들어오면 출력으로 나오게 만들면됩니다.

예시로
1000번 같은 경우는 파이썬으로는 
```python
a, b = input().split()
print(int(a) + int(b))
```

이렇게 풀립니다.
입력이 `3 1` 이런식으로 공백을 사이로 두고 주어지니 `input()`으로 한줄을 받고 `.split()`으로 공백을 기준으로 두개로 나누고 `a`, `b`에 각각 담습니다.

그 다음 int로 각각 변환(`int()`를 씌움) 하고 더해서 `+` 출력해주면 됩니다. (`print()`)

응용해서 1001도 풀 수 있습니다.
